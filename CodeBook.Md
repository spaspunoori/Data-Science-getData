### Code Book

Code Book that describes the data been used, variables,and any transformations or work that you performed to clean up the data.

Data been organized into two sets "TEST" and "TRAIN"

## Each set contains

    1. measurements from the accelerometer and gyroscope
    2. activity label
    3. identifier of the subject

The set of variables that were estimated from these signals are:

    mean: Mean value
    std: Standard deviation
